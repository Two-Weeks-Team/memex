Qdrant Vector Space Day 2026 — Think Outside the Bot

Your session history, as a memory you can move through.

You don't document every thought. Yet when the moment is right, the relevant one comes back on its own. Memex does that for your coding sessions: it keeps the corpus you own and lets you move through it by similarity, not keywords.

Built over your Claude Code / Codex history. No chatbot. No LLM at runtime. Vectors live in Qdrant; you navigate the space.

Not a chatbot. A place your past work stays reachable.

Claude Code silently migrates and deletes old sessions. Memex preserves them in a corpus you own, then turns it into a navigable spatial / associative memory.

What it is not

  • Not a search box you query with keywords
  • Not a chatbot answering on your behalf
  • Not calling an LLM at runtime

What it is

  • An associative trail through your own history — Vannevar Bush's 1945 Memex idea
  • Reads ~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl
  • Embeddings indexed in Qdrant; you move by similarity

The surfaces

Different ways to move through the same memory space.

Time Machine

Replay sessions as they actually unfolded.

Lens

Named-vector weighted search — bias the space toward what you mean.

Topology

Distance matrix turned into an MST graph of related work.

Mix / Discovery

Positive / negative recommendation — more like this, less like that.

Proactive Recall

Relevant past context surfaces as your current work approaches it.

Dashboard

One overview of the corpus and where you've been.

Companion / Wrapped

The agent-memory layer — a longer view of how your sessions evolve.

Loop Breaker

Spot when you're circling the same problem and step out of it.

Run it

Two ways. The server / Docker variant bundles Qdrant, the web UI, and the MCP server in one image — nothing exotic to install.

Docker (server variant)

# clone
git clone https://github.com/Two-Weeks-Team/memex
cd memex

# Qdrant + web + MCP in one image
docker compose up -d

# open the web UI
open http://localhost:8080

Local

# index your existing session history
memex index ~/.claude/projects

# start the web surfaces
memex web

# then browse Time Machine, Lens,
# Topology, Mix, Recall…

For agents — one line to wire it in

Memex exposes 11 MCP tools over stdio + HTTP. It works inside Claude Code, Codex, and Cursor — your agent reaches into your own memory instead of starting cold. 

claude mcp add memex -- memex mcp
time_machine lens_search topology mix_discover recall dashboard companion wrapped loop_breaker + 2 more
