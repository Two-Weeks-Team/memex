No LLM at runtime. No round-trip to recall.

Memex is a non-chatbot. Surfacing is vector math against a local store — not a model generating text on a server.

  • Recall is computed from embeddings already on disk — no prompt, no completion call.
  • Because there's no runtime model dependency, it works offline and stays predictable.
  • The corpus exists because Claude Code silently migrates and deletes old sessions — Memex preserves a copy you own from ~/.claude/projects/**/*.jsonl and legacy transcripts.
