Text alternative: what the map shows

Each node is one past coding session, placed by vector similarity. Edges are the minimum spanning tree over the session-to-session distance matrix — the shortest set of links that keeps every session connected to its nearest neighbours. The graph below is an illustrative sample, not live data from your machine:

    Six lenses, one corpus

    Every session is embedded under several named vectors. The Lens surface lets you re-weight them, so the same history can mean different things depending on what you’re looking for. Weights below are an example mix — yours are whatever you set.

    Re-weighting changes which neighbours are “closest,” so the topology you walk through changes too.

    Why it exists

    Inspired by Vannevar Bush’s 1945 Memex and its associative trails.

    Your history is fragile

    Claude Code can silently migrate or delete old sessions. Memex preserves a corpus you own.

    Keywords aren’t how you think

    You move by association. Memex lets you travel by similarity instead of exact text match.

    No model at query time

    Embeddings live in Qdrant; navigation is LLM-free at runtime, so it runs local-first.
