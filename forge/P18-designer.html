What it is

A corpus you own, navigated by similarity — not keywords.

Claude Code quietly migrates and deletes old sessions. Memex reads your local history (~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl) and turns it into a navigable spatial memory backed by Qdrant. No model runs at query time — the work is done in vectors.

“…that auth refactor from three weeks ago — what did past-me decide?”

You move through the corpus the way a memory returns: by proximity and association, following a trail from one fragment to the next.

For your agent

11 MCP tools, over stdio + HTTP.

Memex exposes its memory to agents directly. It works in Claude Code, Codex, and Cursor — so the assistant can reach back into your shared history mid-task.

# add Memex as an MCP server in Claude Code
claude mcp add memex -- <memex stdio command>

Eleven tools cover recall, lens, topology, and the agent-memory layer. See the repo for the exact command and tool list.
