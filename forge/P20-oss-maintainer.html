A memory you own — that resurfaces on its own.

You don't document every memory. Yet mid-task, the relevant thought arrives unbidden. memex turns your Claude Code / Codex session history into a navigable associative memory — and surfaces the past to you, by the situation, not by a search box.

Apache-2.0, local-first, no runtime LLM, no lock-in. The corpus stays yours, preserved against silent session migrations.

What memex actually is

InputYour local ~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl. Nothing leaves your machine to make it work.
ModelEmbeddings into a Qdrant vector space. No LLM call at runtime for recall — it's distance, not a chatbot.
SurfacesTime Machine (replay), Lens (named-vector weighted view), Topology (distance-matrix → MST graph), Mix/Discovery, proactive Recall, Dashboard, and the Companion / Wrapped / Loop Breaker agent-memory layer.
WhyClaude Code silently migrates and deletes old sessions. memex preserves a corpus you own and lets you move through it by similarity, not keywords.

Get it running

Two paths. Both keep your data on your machine.

Local clone

$ git clone https://github.com/Two-Weeks-Team/memex
$ cd memex
# indexes your existing Claude Code / Codex history into a local Qdrant
$ make up   # brings up Qdrant + web + MCP

Server / Docker variant — one image

Qdrant + web UI + MCP server bundled together, for a homelab or a shared box you control.

$ docker run -p 8080:8080 -v ~/.claude:/data/claude:ro memex/server
# web UI on :8080, MCP exposed over HTTP + stdio

Wire it into your agent

11 MCP tools over stdio + HTTP. Works in Claude Code, Codex, and Cursor.

$ claude mcp add memex -- memex serve --mcp
  • recall
  • lens.search

  • mix.discover

Surfaces

Proactive Recall

The past rises to meet the present — the surface shown above. Driven by your current context, not a query.

Topology

Distance matrix → minimum spanning tree. The shape of your corpus as an associative trail map.

Lens

A named-vector weighted view: lean the space toward a facet without typing a search.

Time Machine

Replay a session as it unfolded — the record Claude Code would have quietly migrated away.

Mix / Discovery

Positive/negative recommendation across the corpus. "More like this, less like that."

Companion / Wrapped / Loop Breaker

The agent-memory layer: a running companion, a periodic Wrapped, and a check against repeating yourself.

Why "community-owned" is the point

Your session history is the most honest record of how you actually think and build. A vendor can migrate it, deprecate it, or delete it on their schedule. memex makes a copy you control.

The recall is autonomous, but the corpus is yours: Apache-2.0 source, a local Qdrant you can inspect, plain .jsonl you can read with cat. No account, no runtime model dependency, no silent migration. If the project went away tomorrow, your fork keeps working.
