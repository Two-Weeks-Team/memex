What it actually is

A small tool over your session history (~/.claude/projects/**/*.jsonl + legacy transcripts). It turns that history into a navigable, associative memory backed by Qdrant — so the corpus is yours and you move through it by similarity, not keywords.

01 Proactive Recall

The old session surfaces because the current context evoked it — not because you went looking. The motif above, for real work.

02 Lens & Topology

Named-vector weighted views and a distance-matrix MST graph — to wander your past by closeness instead of filenames.

03 Time Machine

Replay a session you'd otherwise lose. Claude Code silently migrates and deletes old sessions; Memex keeps the corpus.

04 Companion / Wrapped / Loop Breaker

An agent-memory layer that notices when you're circling the same problem — and brings back what already worked.

Run it the scrappy way

No SaaS account, no runtime LLM bill. Self-host the whole thing in one Docker image (Qdrant + web + MCP), or wire it straight into your agent.

Server / Docker variant

# Qdrant + web UI + MCP in one image
git clone https://github.com/Two-Weeks-Team/memex
cd memex
docker compose up -d
# then open the web UI & index your history

Wire it into your agent (MCP)

# 11 MCP tools over stdio + HTTP
claude mcp add memex -- memex-mcp
# works in Claude Code / Codex / Cursor
# now Recall can surface mid-task, autonomously
