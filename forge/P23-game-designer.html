What Memex actually is

A non-chatbot tool, with no LLM call at runtime, that reads your Claude Code / Codex session history (~/.claude/projects/**/*.jsonl plus legacy transcripts and history.jsonl) and turns it into a navigable spatial memory backed by Qdrant. It exists because Claude Code silently migrates and deletes old sessions — this keeps a corpus you own.

Wire it into your agent

Memex exposes 11 MCP tools over stdio and HTTP, so it works inside Claude Code, Codex, and Cursor. A server / Docker variant bundles Qdrant + web UI + MCP into one image. 

# add the Memex MCP server to Claude Code
claude mcp add memex -- path/to/memex-mcp
# 11 tools become available: lens search, topology, recall, mix, time-machine…
