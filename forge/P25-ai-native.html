Your agent stops mid-task and remembers.

You didn't search. You didn't paste in context. While your coding agent works, a thought from a session weeks ago rises into its window on its own — because the moment called it up.

Honest twist: Memex is LLM-free at runtime. It doesn't think for your agent — it remembers for it. Your past sessions become a navigable memory in Qdrant, and Memex hands the relevant fragment back to the agent through MCP, unbidden, when the situation matches.

What's actually underneath

A non-chatbot tool over your ~/.claude/projects/**/*.jsonl history (plus legacy transcripts and history.jsonl). It preserves a corpus you own — Claude Code quietly migrates and deletes old sessions; Memex keeps them and lets you move by similarity, not keywords.

Connect it to your agent

Memex exposes 11 MCP tools over stdio + HTTP. Add it once; recall then arrives inside Claude Code, Codex, or Cursor without you reaching for it. 

# register Memex as an MCP server in Claude Code
claude mcp add memex -- npx -y @two-weeks-team/memex-mcp
# or run the all-in-one server variant (Qdrant + web + MCP in one image)
docker run -p 8080:8080 -v ~/.claude:/data/.claude ghcr.io/two-weeks-team/memex:latest
