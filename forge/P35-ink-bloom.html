This page shows coding memories that surface automatically as the context changes. A reduced-motion setting shows a single memory rendered statically. The full project is described in the section below.

Not a search box. Not a chatbot.

Claude Code silently migrates and deletes old sessions. Memex preserves a corpus you own — your ~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl — and lets you move through it by similarity rather than keywords. It runs LLM-free at query time, backed by Qdrant.

Wire it into your agent

11 MCP tools over stdio + HTTP. Works in Claude Code, Codex, and Cursor. A server/Docker variant runs Qdrant + web + MCP in one image.

# add the Memex MCP server (stdio)
claude mcp add memex "memex serve --mcp"

# or run the all-in-one server variant
docker run -p 8080:8080 two-weeks-team/memex
