What's actually happening

Memex is a non-chatbot tool that runs over your Claude Code and Codex session history. No LLM at runtime — it turns the transcripts you already own into a navigable, associative memory you can move through by similarity, not keywords.

Your corpus, preserved

Reads ~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl — a corpus you keep, even when old sessions get silently migrated or removed.

Backed by Qdrant

Embeddings live in a Qdrant vector space, so recall is by closeness of meaning. A server / Docker variant runs Qdrant, the web UI, and MCP together in one image.

Surfaces, not searches

Time Machine replay, Lens weighted vectors, Topology graphs, Mix/Discovery, and a proactive Recall that brings a past moment forward when the current one rhymes with it.

Agent-memory layer

A Companion / Wrapped / Loop-Breaker layer reflects on patterns across your sessions — what you keep circling back to, and where you've been before.

Plug it into your agent

11 MCP tools over stdio + HTTP — works in Claude Code, Codex, and Cursor. Your agent can recall, replay, and connect across your own history.
