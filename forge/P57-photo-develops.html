The facts

A memory you own — not a chatbot.

Claude Code silently migrates and deletes old sessions. Memex preserves that corpus and lets you move through it by similarity. LLM-free at runtime.

Your own history, indexed

Reads ~/.claude/projects/**/*.jsonl, legacy transcripts, and history.jsonl — then turns it into navigable spatial memory backed by Qdrant.

Surfaces, not searches

Time Machine, Lens (named-vector weighting), Topology (MST graph), Mix/Discovery, proactive Recall, plus a Companion / Wrapped / Loop Breaker layer.

Lives in your agent

11 MCP tools over stdio + HTTP — works in Claude Code, Codex, and Cursor. A server/Docker variant runs Qdrant + web + MCP in one image.
