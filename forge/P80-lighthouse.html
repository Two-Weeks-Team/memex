what the lighthouse actually is, below

Why a lighthouse

You don't document every session. The relevant one still surfaces — on its own.

Memex is a non-chatbot, LLM-free-at-runtime tool that reads your Claude Code / Codex history (~/.claude/projects/**/*.jsonl, legacy transcripts and history.jsonl) and turns it into a navigable spatial memory backed by Qdrant. Each past session sits somewhere out in the dark. The beam sweeps by similarity to what you're doing now — so retrieval happens to you, not because you went looking.

🌊

A sea you own

Claude Code silently migrates and deletes old sessions. Memex keeps the corpus on your machine, yours to move through by meaning — not by keyword.

🔦

The beam, not your cursor

There's no search box and nothing to hover. The light rotates by itself; when its angle crosses a relevant moment, that scene is caught and held — then released.

🧭

Vannevar Bush's Memex

The 1945 idea: follow associative trails through what you've already thought, instead of querying a database. Distance in the dark is the index.

For your agent

11 MCP tools your agent can reach into mid-task.

Memex exposes 11 MCP tools over stdio + HTTP and works in Claude Code, Codex, and Cursor. A server / Docker variant runs Qdrant + the web UI + MCP in one image, so the same associative sea is reachable by you and by the agent — the beam can be triggered by the moment your agent is in, not by a query.
