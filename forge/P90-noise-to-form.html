What Memex actually is

A memory you own — not a chatbot, no LLM at runtime.

Claude Code silently migrates and deletes old sessions. Memex preserves that corpus locally and makes it navigable by association, the way Vannevar Bush imagined the original Memex in 1945.

[01]Reads your real history

Indexes ~/.claude/projects/**/*.jsonl plus legacy transcripts and history.jsonl — the sessions you actually ran.

[02]Backed by Qdrant

A local vector space turns history into spatial memory. Five distinct Qdrant primitives are the product — not a sidecar.

[03]No chatbot, LLM-free at runtime

Nothing phones an LLM to recall. You move through the corpus by similarity — keywords optional.

[04]Runs as desktop app or server

A server / Docker variant runs Qdrant + web UI/API + MCP in one image.
