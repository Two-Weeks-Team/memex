Brief Abstract/Summary of the Project (What problem does it solve and how?)*

Past-you already solved this. The moment your coding agent loops on a problem, Memex resurfaces the past session that fixed it — no chat, no search box. Memex is a local memory layer for Claude Code and Codex: it indexes every past session into Qdrant 1.18 and turns your history into eight surfaces, zero chat windows — a 3D Time Machine card deck, a force-directed Topology galaxy (Distance-Matrix API) with cross-project bridge + gap insights, Mix & Match recommendations (Discovery API), proactive error recall (dedicated error named vector + has_errors filter), next-action prediction (neighbor pivot-walk + tool-call aggregation), turn-by-turn Replay, a weighted multi-named-vector Lens, and — through a 12-tool MCP server — an agent-memory surface that drops the right past session into the model's own turn. It matters because Claude Code silently rewrites its storage and auto-updates delete old transcripts (dozens of OPEN data-loss reports, none acknowledged in the changelog); a portable Qdrant snapshot puts that corpus permanently in your hands. Five distinct Qdrant primitives are load-bearing, not a sidecar, and the AI only embeds — zero LLM at runtime, zero network calls, 100% local, in Rust + Tauri 2 + Qdrant. Vannevar Bush's 1945 memex ran on associative trails, not search boxes; a vector database finally makes that literal.
